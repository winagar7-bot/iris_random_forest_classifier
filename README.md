#  Random Forest Classifier on Iris Dataset

## 📌 Project Overview
This project demonstrates how to train and evaluate a **Random Forest Classifier** using the classic **Iris dataset**.  
The notebook provides a step-by-step workflow:
- Load the dataset
- Split into training and testing sets
- Train a Random Forest model
- Evaluate performance
- Run inference on new samples

## Files
- `random_forest_iris.ipynb` → Jupyter Notebook containing the full workflow
- `README.md` → Documentation for setup and usage
